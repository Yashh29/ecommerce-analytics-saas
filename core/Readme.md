# End-to-End E-Commerce Analytics & Customer Intelligence Platform

This project demonstrates an industry-style end-to-end data science workflow for an e-commerce business.

## Key Features
- Customer Segmentation (K-Means)
- Customer Lifetime Value (CLV) Prediction
- Churn Prediction & Risk Scoring
- Executive & Operations Dashboard (Streamlit)

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Matplotlib

## Dashboard
The Streamlit dashboard provides:
- Executive KPIs
- Segment-wise churn insights
- CLV distribution
- Churn risk analysis and customer prioritization

## Note
The data used in this project is anonymized and synthetically generated to simulate real-world e-commerce patterns.
